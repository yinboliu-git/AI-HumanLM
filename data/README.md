# Data Directory

Due to size constraints, raw data is not included in this repository.

## Download Data

- **Moltbook Dataset**: https://huggingface.co/datasets/qugemingzi/moltbook-ai-agent-posts
- **Reddit Data**: Available via PRAW API (see src/crawl_reddit.py)
- **OASST Data**: https://huggingface.co/datasets/OpenAssistant/oasst1

## Processed Data

After running the pipeline, processed data will be stored in:
- `data/processed/aligned_corpus.json`
- `data/processed/metrics_raw.csv`
- `data/processed/statistical_tests.csv`
